Klucz API Gemini

Potrzebujesz klucza API, aby korzystać z funkcji skanowania paragonów

Jak uzyskać klucz API?

1
Przejdź do Google AI Studio
Otwórz stronę aistudio.google.com
2
Uzyskaj klucz API
Po lewej stronie na dole kliknij "Get API key" lub przejdź bezpośrednio do aistudio.google.com/apikey
3
Utwórz nowy klucz
Kliknij "Create API key" na górze strony, wybierz projekt (jeśli nie masz projektu, najpierw go utwórz)
4
Skopiuj klucz
Skopiuj wygenerowany klucz API i wklej go w polu poniżej
Twój klucz API jest przechowywany bezpiecznie i używany wyłącznie do analizy paragonów

Prześlij plik z dysku

Przeciągnij i upuść paragon lub wybierz plik z urządzenia

Zrób zdjęcie kamerą

Użyj kamery urządzenia aby zrobić zdjęcie paragonu

Wskazówki skanowania

  • Rozłóż paragon na płaskiej powierzchni i zadbaj o równomierne oświetlenie bez cieni.
  • Ustaw aparat prostopadle do paragonu, aby uniknąć zniekształceń i rozmycia.
  • Przytnij kadr tak, by wypełniał go sam paragon – przyspiesza to analizę obrazu.
